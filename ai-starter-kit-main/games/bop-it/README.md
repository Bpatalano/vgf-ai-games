# Bop-It! Voice Game

A modern web-based version of the classic Bop-It! game with voice recognition.

## How to Play

1. Click "Start Game" to begin
2. Listen for voice commands like "BOP IT!", "TWIST IT!", "PULL IT!"
3. Respond quickly by saying the command word (e.g., "BOP", "TWIST", "PULL")
4. Speed increases with each successful round
5. Try to beat your high score!

## Voice Commands

- **BOP IT!** ’ Say "BOP"
- **TWIST IT!** ’ Say "TWIST"
- **PULL IT!** ’ Say "PULL"
- **FLICK IT!** ’ Say "FLICK"
- **SPIN IT!** ’ Say "SPIN"
- **PASS IT!** ’ Say "PASS"

## Requirements

- Modern web browser with microphone access
- Microphone permission required for voice commands
- Internet connection for initial load

## Development

Built with vanilla JavaScript, HTML5, and CSS3. Uses Web Speech API for voice recognition.
